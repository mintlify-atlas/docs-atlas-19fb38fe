## Key Features

<CardGroup cols={2}>
  <Card title="Case Management" icon="folder" href="/features/case-management">
    Create and organize PDF collections with automated ingestion to Hugging Face datasets
  </Card>
  <Card title="Visual Search" icon="magnifying-glass" href="/features/visual-search">
    Vector-based search using ColPali embeddings stored in LanceDB
  </Card>
  <Card title="AI Q&A" icon="messages-question" href="/features/ai-qa">
    Question-answering on document images powered by Qwen2-VL
  </Card>
  <Card title="No OCR Required" icon="ban" href="/architecture/overview">
    Process documents visually without traditional text extraction
  </Card>
</CardGroup>

## Quick Start

<Steps>
  <Step title="Set up prerequisites">
    Configure Supabase for authentication and Modal for LLM serving
  </Step>
  <Step title="Deploy the application">
    Use Docker Compose or run the backend and frontend separately
  </Step>
  <Step title="Create your first case">
    Upload PDFs to create a searchable collection
  </Step>
  <Step title="Search and explore">
    Use visual search and AI Q&A to explore your documents
  </Step>
</Steps>
